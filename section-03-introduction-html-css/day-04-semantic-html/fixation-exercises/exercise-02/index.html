Fatos sobre o Stomatopoda

Descrição

O Stomatopoda conhecido cientificamente como Odontodactylus scyllarus é uma ordem de crustáceos marinhos da subclasse Hoplocarida, que agrupa cerca de 400 espécies, caracterizadas principalmente pela morfologia da segunda pata torácica, que é modificada em apêndice subquelado, lembrando uma pata de louva-a-deus.
